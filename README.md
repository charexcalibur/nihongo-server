# nihongo-server

本项目为nihongo-back及nihongo项目，提供后端服务。

本后端项目主要使用使用以下技术栈开发:

- nodejs 6.9.1

- express 4.15.2

- mongoose 4.13.7

## 使用

```bash
# install dependencies
npm install

# run
npm run start

```

## 功能

目前实现了以下功能:

- 后台用户登录

- 后台用户注册

- 后台重复用户id检测

- 题目录入

- 题目录入重复id验证

- 题目查询

- 题目删除

- 题目查询时编辑
